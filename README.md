# Flavourly - Recipe Finder Web App

Flavourly is a modern, user-friendly recipe finder web application that helps users discover new recipes, search by ingredients or names, and save their favorite meals. The app focuses on simplicity, clarity, and an enjoyable browsing experience.

---

## üé® Design Rationale

The design of Flavourly was created with the following goals:

- **Search-first approach:** The homepage features a prominent search bar and "Surprise Me" button, encouraging users to find recipes immediately.
- **Visual clarity:** Cards, badges, and hero sections use contrasting colors and shadows to guide attention to important content.
- **Readability:** Clear typography and spacing make instructions, ingredients, and recipe details easy to read.
- **Mobile-first responsiveness:** Components adjust layout for smaller screens to maintain usability across devices.

- **Search recipes** by name or ingredient.
- **Random recipe suggestion** with the "Surprise Me" feature.
- **Recipe details page** with instructions, images, ingredients, and external links (YouTube or recipe source).
- **Save favorite recipes** and manage them.
